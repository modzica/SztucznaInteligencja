# Przykładowe prompty

## Prompt 1 — analiza kodu
```text
Przeanalizuj ten kod Python. Wyjaśnij, co robi, wskaż błędy i zaproponuj poprawioną wersję. Zachowaj ten sam cel funkcji.
```

## Prompt 2 — testy
```text
Napisz testy jednostkowe w pytest dla tej funkcji. Uwzględnij poprawne dane, błędne dane i co najmniej jeden edge case.
```

## Prompt 3 — refaktoryzacja
```text
Zrefaktoryzuj ten kod Python, aby był bardziej czytelny i łatwiejszy do testowania. Nie zmieniaj działania funkcji.
```
